# CacheSense

CacheSense is a lightweight cache analytics wrapper for Node.js libraries.

## v1 Goals

- Track cache hits
- Track cache misses
- Report hit ratio
- Expose a stats endpoint
- Provide a simple Express example

## Project Structure

- `src/CacheSense.js`
- `src/Metrics.js`
- `src/StatsServer.js`
- `examples/express-example.js`
- `examples/node-cache-example.js`
